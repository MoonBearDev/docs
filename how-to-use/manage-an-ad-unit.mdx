# 광고 단위 관리하기

DARO 대시보드의 Ad Units 뷰에서, 추가한 광고 단위를 살펴보고 관리할 수 있습니다.

### 상세 정보 안내

- Ad Unit Name: 광고 단위를 추가할 때 작성한 광고 단위의 이름입니다. Ad Unit Name은 실제로 연동할 때 사용하진 않지만, 대시보드에서 데이터를 분석하거나 광고 단위를 구분할 때 사용되는 구분자라고 할 수 있습니다.
- Ad Unit ID: 플랫폼에서 구분하는 광고 단위 구분자이며, SDK를 연동할 때 활용됩니다.
- Ad Format: 광고 단위를 추가할 때 선택한 광고 형태입니다.
- Status: 광고 단위의 상태 값을 나타냅니다.
    - In Review: 기본적으로 Live 상태로 바로 생성되나 이슈가 발생한 경우 In Review 상태에 놓여질 수 있습니다. 일반적으로 영업일 1일 이내 해결됩니다.
    - Live: 정상적으로 광고 단위를 활용할 수 있는 상태입니다. AD UNIT ID를 가이드에 맞게 각 지면에 구현하세요.
        
        💡참고: 광고 단위는 각 유형별로 총 20개만 Live 상태로 활용할 수 있습니다.
        
    - Pause: 광고 단위 정지 버튼을 통해 활용을 정지한 상태입니다. 앱에 적용했던 AD UNIT ID가 있다면 광고가 송출되지 않기 때문에 제거해주시기 바랍니다.
- Action: Edit 버튼을 통해 광고 단위의 설정 값을 변경할 수 있습니다. 이미 생성된 키값과 광고 형태는 변경할 수 없으나, 광고 단위 이름 과 최소 eCPM을 변경할 수 있습니다.
    
    💡참고: 최소 eCPM은 fill rate에 영향을 줄 수 있기 때문에 기본적으로는 Off 상태로 두는 것을 권장합니다.
