# 왜 오늘과 1일 전 데이터는 확인할 수 없나요?

DARO 대시보드에서는 광고가 노출되고 지표가 기록되기까지 2일이 소요됩니다. 다양한 광고 네트워크들의 데이터를 수집하고 가공하여 Daro 대시보드에 적용하기까지 시간이 소요되기 때문입니다.

많은 광고 네트워크들이 실시간으로 데이터를 전달하려고 노력하고 있으나, 부정 클릭과 같이 추후에 변경되는 값들이 있기 때문에 부득이하게 업데이트하는데까지 1일 이상이 걸리는 경우도 있습니다. 따라서, DARO에서는 이런 모든 데이터를 받아서 안정적으로 대시보드에서 제공하기까지 2일이 소요되는 점 양해 부탁드리겠습니다.

혹시나 SDK 연동 테스트와 같이 실제 지표가 기록되는지 정밀하게 살피셔야 하는 경우에는 [support@daro.so](mailto:support@daro.so) 메일로 문의해주시기 바랍니다.
